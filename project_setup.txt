# AlphaForge

AI-Powered Strategy Intelligence & Quant Research Platform

---

# Vision

AlphaForge is a production-grade AI agentic platform designed to:

* Ingest trading strategies from PDFs, Telegram, YouTube, TradingView, and notes
* Convert human trading concepts into deterministic logic
* Generate Pine Script and Python strategies automatically
* Backtest strategies using vectorbt
* Analyze performance across multiple market regimes
* Produce AI-generated insights and optimization suggestions
* Build a robust quantitative research workflow

This is NOT a signal-selling bot.

This is:

* an AI systems engineering project
* a quantitative research platform
* an institutional-style strategy intelligence engine

---

# High-Level Architecture

```text
Telegram / PDFs / YouTube
            ↓
Strategy Ingestion Agent
            ↓
Structured Strategy Schema
            ↓
Rule Compiler Agent
            ↓
Pine + Python Strategy Generator
            ↓
Backtesting Engine (vectorbt)
            ↓
Evaluation + Insight Engine
            ↓
Dashboard + Analytics
```

# Environment Setup

## 1. Install Python

Recommended:

* Python 3.11+

Verify:

```bash
python --version
```

---

# 2. Install uv (Recommended Package Manager)

Mac/Linux:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Verify:

```bash
uv --version
```

---

# 3. Create Project

```bash
mkdir alphaforge
cd alphaforge
```

---

# 4. Initialize Git

```bash
git init
```

---

# 5. Create Virtual Environment

```bash
uv venv
```

Activate:

Mac/Linux:

```bash
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

---
